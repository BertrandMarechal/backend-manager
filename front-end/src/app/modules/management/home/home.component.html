Welcome to the management application

This application is aimed to help with my (and maybe your) coding process. It works on an infrastructure based on AWS Lambda functions calling a Postgres server.

Pre-requisites

In order for this application to work, the file structure is the following : - The database folders should be suffixed with an identifiable suffix editable in settings
- The middle tier folders should be suffixed with an identifiable suffix editable in settings
- This application should run at the same level as your repositories
serverless.yml
variable.yml
version folders and version.json
